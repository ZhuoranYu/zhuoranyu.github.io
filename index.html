    Projects

  • Error Detection via Low Rank Approximation

    • We are trying to detect potential errors in structured tables. Traditional rule-based methods require users to explicitly specify detection rules, which is ad-hoc and costs a lot of human labor. In this project, we provide a data-driven approach which makes the use of functional dependencies between columns in data tables to detect errors. We develop a robust low rank approximation model to capture the general information of data and use this information to generate probability of each cell being erroneous.

  • SenseKitchen: Anomaly Detection in Kitchen for Food Safety Guard

    • The main objective of this project is to build a detector which detects anomaly behavior of chiefs in kitchens. Algorithmic solution is composed by two phases: object detection which detects human heads and body and classification which classifys if the behavior is anomalous. In the latest release, object solution phase achieves 5-times faster inference than previous releases and all associated computation can be done one client device.

  • Stream Graph Partitioning

    • Social network graphs usually grow too big to be stored in a single storage node. In this project, we evaluate the performance of several stream partitioning algorithms in response to different types of queries in terms of both workload balance and responce time.
